# anshu957.github.io

My personal site — writing, research, a CV, and too much tea. Astro + a bit of React.

```bash
npm install
npm run dev      # local preview
npm run build    # static output in dist/
```

Push to `main` and GitHub Actions builds and publishes to [anshu957.github.io](https://anshu957.github.io/).

**Where things live:** posts in `src/content/blog/`, site copy in `src/data/site.js`, about page in `src/components/manuscript/ManuscriptAbout.astro`, styles in `src/styles/manuscript.css`. CV PDF goes in `public/assets/cv/` (path in `src/data/cv.js`).

Design notes for future-me: `AESTHETICS.md`, `CORE_UI.md`.
