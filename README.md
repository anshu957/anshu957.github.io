# MyWebpage

Personal website built with Astro, React islands, and Motion.

## Stack

- Astro for the site shell and routing
- React for interactive islands
- Motion for animated UI elements

## Commands

```bash
npm run dev
npm run build
npm run preview
```

## Structure

- `src/layouts` shared Astro layouts
- `src/components` reusable page components and interactive islands
- `src/animations` local animation data used by React islands
- `src/data` shared homepage content
- `src/styles` design tokens and global styles
- `public` static assets served as-is
- `assets/soundwave-icons` source icon pack reserved for future blog use
