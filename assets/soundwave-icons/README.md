# Soundwave Icons

80 tiny 8-bit sound-wave SVGs on an 8x8 grid.

## Contents

- Files: `soundwave-001.svg` through `soundwave-080.svg`
- Manifest: [manifest.json](./manifest.json)
- Generator: [generate.mjs](./generate.mjs)

## Families

- bars: Bar Wave
- mirror: Mirror Wave
- pulse: Pulse Wave
- ripple: Ripple Echo
- speaker: Speaker Wave
- beam: Beam Wave
- stereo: Stereo Wave
- spark: Spark Wave
- oscillator: Oscillator Wave
- wavelet: Wavelet
- echo: Echo Wave

## Usage

Import the SVGs directly where you need them. They are designed to scale cleanly and stay legible at small sizes.
